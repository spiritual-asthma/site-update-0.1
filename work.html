Work

Sun Orb

Sun orb is an interactive 3-D rendering modelled after an optical phenomenon known as a 22° halo. The scene is animated and can be manipulated by clicking and dragging it with your cursor. Hemisphere lighting was added to the scene to add a beautiful gradient effect to the reflective geometries in the scene.
This rendering was made using with HTML5 Canvas, CSS and Three.js library.

Just Desserts

Just Desserts is a small business bakery website designed for intuitive and informative use by customers. Sleek, light and responsive elements allow users from all platforms to get a consistent virtual landscape of a local treasure. Parallax scrolling was added to the hero image to create an illusion of depth and immerse the user into the virtual experience.
This website was created using Adobe Illustrator, HTML, CSS and Javascript.

Calculator

The Calculator web app can preform small-scale operations for simple day-to-day calculations. This calculator was created using HTML tables and the Javascript onClick method to display clicked table data cells onto a display screen. After evaluating the inputed expression the calculator will output the correct result onto the display screen. The calculator was designed with CSS with dreamy gradients and defining drop shadows.

Countdown Landing Page

The Countdown Landing Page is a dynamic page designed to generate an exciting experience for the user. A timer created using JavaScript takes in a specified date (June 1st 2019) and at a 1 second interval counts down from the current second until the specified date. A form element was added to prompt the user for email updates. Finally, motion graphics using Three.js library were added to keep the page engaging.

Snow in the City

Snow in the City is an active scene built with the particles.js library. It uses HTML and CSS to render the image of New York City’s Time Square as a background image. Particles.js was used to create a snow fall effect to compliment the photo. With the library I was able to manipulate the direction, speed and size of the particles.

Ironic Mad-libs

Ironic mad-libs is an entertaining game created using PHP. It takes the lyrics of Alanis Morissette's “Ironic” and uses the shuffle function to re-arrange the nouns, verbs and adjectives around with every refresh. The result is always a new, yet familiar song with very unusual and entertaining lyrics.
